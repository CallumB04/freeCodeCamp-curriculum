Introduction to Python

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

Not surprisingly, given its accessible and versatile nature, Python is among the top five most popular programming languages in the world. Python is used by Wikipedia, Google, Yahoo!, CERN and NASA, and many more world-famous organisations.

Some of the most-common uses for Python includes:

  • AI & Machine Learning
  • Data Analytics
  • Web Development
  • Search Engine Optimisation
  • Blockchain
  • Game Development
  • Automation
Hello world

To get started with Python, we will write a message to the console:

print("Hello, world!")

The print() function allows for the output of a value to the console, in this line of code we are writing "Hello, World".

Data Types

In Python there are many data types, each with their own purpose and uses:

## "Hello, World" -> String (A concatenation of letters, numbers and/or symbols)
## 8 --------------> Integer (A whole number)
## 2.85 -----------> Float (A decimal point number)
## True/False -----> Boolean (Binary value. True/False or 1/0)
## 'A' ------------> Character (Singular letter/number/symbol value)
Variables

One way we hold data in Python is through the use of variables. A variable holds a singular value of a specific data type and can be referenced and altered throughout a program. In python, variables need to be assigned a value when they are created, even if this value happens to be empty.

Examples of variables:

name = "Callum"
age = 20
pi = 3.14
raining = True
grade = "A"

There are also many different ways of writing variables, the two most commonly used are:

  • Camel Case -> isRainingToday = False... All words capitalised aside from first word.
  • Snake Case -> first_name = "Noah"... All lower-case, underscore seperating words.
Conditions

In Python, there are a built-in list of keywords that all serve different functions. One of them is the if keyword. This allows for logical operations through something known as conditional statements:

if (condition):
do this

elif (another condition):
do this instead

else:
do this if no condition was True

A condition doesnt always need to have an 'elif' or 'else' statement, it can just be a singular condition. However, this allows for multiple possibilities to be checked at once. Below are a few examples:

  • Singular condition:

    weather = "raining"

    if weather == "raining": ## == -> 'is equal to'
    print("It's raining!")

  • If + else:

    age = 18

    if age < 18:
    print("You cannot drink alcohol!")

    else: ## age is not less than 18
    print("You are allowed to drink alcohol!")

  • If, elif + else:

    age = 21

    if age < 18:
    print("You are too young to gamble!")

    elif age >= 18 and age < 25: ## >= -> 'bigger OR equal to' ... and -> checks multiple conditions are both True
    print("You need to show ID!")

    else:
    print("You can gamble without any issues!")
